# my-first-repo

はじめてのGitHubリポジトリです。

## 概要

このリポジトリはGitHubのPR（プルリクエスト）の練習用です。

## 使い方

1. リポジトリをクローンする

```bash
git clone https://github.com/ikeda-web/my-first-repo.git
```

2. ブランチを作成する

```bash
git checkout -b feature/your-feature
```

3. 変更を加えてコミットする

```bash
git add .
git commit -m "説明的なコミットメッセージ"
```

4. PRを作成する

```bash
gh pr create --title "タイトル" --body "説明"
```
